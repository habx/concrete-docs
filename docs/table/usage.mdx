## Overview

**Concrete** has a table UI implementation based on [react-table](https://react-table.tanstack.com/) hooks.
So be sure to understand how [react-table](https://react-table.tanstack.com/) hooks work before doing anything.

## Installation

**Concrete** table needs other dependencies to work. So first run

```bash
  npm i @habx/ui-table react-table
```

## Basic usage

**Concrete Table** exports an abstraction of `useTable` hook and a component to render the ui of the table.

Here is how it works.

```jsx
import { useTable, Table } from '@habx/ui-table'

export const TableExample = () => {
  const tableInstance = useTable({
    data: [
      { name: 'Jon', age: 19 },
      { name: 'Eddy', age: 30 },
      { name: 'Michel', age: 40 },
    ],
    columns: [
      {
        Header: 'Name',
        accessor: 'name',
      },
      {
        Header: 'Age',
        accessor: 'age',
      },
    ],
  })
  return <Table instance={tableInstance} />
}
```

export const data = [
  { name: 'Jon', age: 19 },
  { name: 'Eddy', age: 30 },
  { name: 'Michel', age: 40 },
]

export const columns = [
  {
    Header: 'Name',
    accessor: 'name',
  },
  {
    Header: 'Age',
    accessor: 'age',
  },
]

export const TableExample = () => {
  const tableInstance = useTable({
    data,
    columns,
  })
  return <Table instance={tableInstance} />
}
